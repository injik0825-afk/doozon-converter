## 🚀 사용 방법

1. 거래내역 엑셀 파일들 다운로드 → Streamlit 앱에 업로드 (여러 파일 동시 가능)
2. **🔄 변환 시작** 버튼 클릭
3. 화면 표시:
   - 🏢 자동 감지된 회사명
   - 📊 전표 행 수, 미분류 건수, 차/대변 합계
   - 🆕 새 종목 감지 시 추가 코드
4. **📥 변환 파일 다운로드** → 색상 적용된 엑셀 파일 받기
5. 더존 위하고 업로드 전 **🔴 빨간 행은 직접 수정** (취득가 등)

## 🔄 GitHub 업로드 방법

### ⚠️ 중요: 복사/붙여넣기 절대 금지

GitHub 웹 편집기에서 코드를 복사/붙여넣기 하면 특수문자(한글, 이모지, 정규식 등)가 깨질 수 있어요. **반드시 파일 자체를 업로드**하세요.

### 작업 순서

1. GitHub `doozon-converter` → `app/` 폴더 → `streamlit_app.py` 클릭
2. 우측 상단 `[...]` → **Delete file** → Commit changes (초록 버튼)
3. `app/` 폴더로 돌아가기
4. 우측 상단 **Add file → Upload files**
5. 다운받은 `streamlit_app.py` 파일 **드래그 앤 드롭**
6. **Commit changes** 클릭
7. 1~2분 대기 → Streamlit Cloud 자동 재배포

## ❓ 문제 해결

### "변환된 데이터가 없습니다" 오류
- 시트명이 인식 가능한 패턴인지 확인
- 빈 파일 또는 형식 깨진 파일 가능성

### 자동 감지가 잘못됨
- 시트명/파일명에 회사 식별 키워드가 부족한 경우
- Claude에게 시트명/파일명 알려주면 감지 패턴 추가 가능

### 센트럴밸류 계정코드가 더존과 다름
- `CENTRAL_AC`는 추정치이므로 첫 사용 시 반드시 더존 계정코드와 대조
- 코드가 다르면 Claude에게 실제 코드 알려주면 수정해 드립니다

### 밸류어블 파일 인식 관련
- 국민은행 시트명이 `고객납입용`으로 변경되어도 자동 감지됩니다
- 비용지출 시트명이 `2026.4월분`, `2026.5월분` 등으로 월이 바뀌어도 자동 감지됩니다
- 미래에셋 시트명이 계좌번호(숫자 10~15자리)이면 자동 처리됩니다
- 교보(`Sheet1`), 대신(`거래내역_전체`), 한화(`myTable`) 시트는 파일명으로 회사 구분합니다
