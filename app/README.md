# 📊 더존 위하고 전표 변환기

거래내역 엑셀 파일을 더존 위하고 일반전표 업로드 형식으로 자동 변환하는 Streamlit 앱.

---

## 🏢 지원 회사 (7개)

| # | 회사 | 자동 감지 키워드 | 계정코드 | 상태 |
|---|---|---|---|---|
| 1 | **리버사이드파트너스** | `IBK 기업은행`, `한국투자증권`, `교보증권`, `비씨카드` | 5자리 | ✅ 11개 일치 |
| 2 | **두리인베스트먼트** | `키움(...)`, `메리츠(...)`, `한국(80...)`, `IBK기업은행 내역` | 3자리 | ✅ 3개 일치 + 🟡 3개 근접 |
| 3 | **엔라이튼자산운용** | `하나은행거래내역_*`, `법인카드승인내역`, `공모주실현손익` | 5자리 | ✅ 3개 일치 + 🟡 3개 근접 |

업로드된 파일의 시트명 + 파일명 보고 회사를 **자동 판단**합니다.

## 🚀 사용 방법

1. 거래내역 엑셀 파일들 다운로드 → Streamlit 앱에 업로드 (여러 파일 동시 가능)
2. **🔄 변환 시작** 버튼 클릭
3. 화면 표시:
   - 🏢 자동 감지된 회사명
   - 📊 전표 행 수, 미분류 건수, 차/대변 합계
   - 🆕 새 종목 감지 시 추가 코드
4. **📥 변환 파일 다운로드** → 색상 적용된 엑셀 파일 받기
5. 더존 위하고 업로드 전 **🔴 빨간 행은 직접 수정** (취득가 등)

## ❓ 문제 해결

### "변환된 데이터가 없습니다" 오류
- 시트명이 인식 가능한 패턴인지 확인
- 빈 파일 또는 형식 깨진 파일 가능성

### 자동 감지가 잘못됨
- 시트명/파일명에 회사 식별 키워드가 부족한 경우
- Claude에게 시트명/파일명 알려주면 감지 패턴 추가 가능

### 로만 운용보수 입금이 🟠 ORANGE로 표시됨
- `증금(로만자산)` 입금은 내역만으로는 미수금/수수료수익 구분 불가
- 분개장 참고해서 직접 계정 입력 필요

### 로만 접대비가 복리후생비로 표시됨
- 카드 명세서 업종만으로는 접대비/복리후생비 구분 불가
- 접대비 건은 더존에서 직접 변경 필요
